Project Overview
GlowDesk AI (also known as Workly Hub) is an elegant, highly intuitive AI-powered productivity assistant designed to streamline daily professional and academic workflows.

Instead of forcing you to constantly switch between a dozen different browser tabs, it brings all your essential workspace tools into one single, beautifully cohesive dashboard. It specifically features a gorgeous, professional feminine aesthetic—built with a soft canvas layout, delicate botanical icons, and a chic dark-berry and pastel rose color palette to balance functional power with design delight.

✨ Key Features
GlowDesk AI Dashboard: A structured main hub featuring dynamic productivity tracking statistics (like hours saved per week and faster response times) to see your efficiency at a glance.

Smart Email Generator: An automation tool that instantly drafts professional, formal, or casual emails based on minimal text prompts.

Notes Summarizer: A workspace where you can drop in long text walls, lecture notes, or meeting transcripts and immediately receive structured, easy-to-read summaries.

Dynamic Task Planner: An interactive calendar and task board to plan out your week, set priorities, and track action items seamlessly.

Research Assistant: A specialized tool designed to handle deep-dives, strategy planning, and complex issue identification effortlessly.

Interactive AI Chat: A dedicated chatbot space for open-ended brainstorming, quick legal or administrative questions, and active task delegation.

🛠️ Tools Used
React & TypeScript (.tsx): The modern development framework used to build a highly scalable, fast, and interactive user interface.

Tailwind CSS: The styling engine managing the custom aesthetic, color variables (like your dark berry text), floral dashboard accents, and responsive card layouts.

Lucide React: A specialized iconography package used to render all the sleek botanical, calendar, and email icons across the sidebar and buttons.

Bun: An ultra-fast, modern JavaScript build tool and package manager used to compile, bundle, and lock down your project files efficiently.

Lovable: The rapid AI-generation platform used to scaffold the initial code components, interface structure, and visual layout.
